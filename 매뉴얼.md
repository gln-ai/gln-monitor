# GLN 카페 모니터링 — 실행 매뉴얼

---

## 준비물 (API 키 3개)

| 항목 | 발급 주소 |
|---|---|
| 네이버 Client ID / Secret | https://developers.naver.com |
| Anthropic API Key | https://console.anthropic.com |
| Gmail 앱 비밀번호 (이메일용) | https://myaccount.google.com/apppasswords |

---

## 1단계 — 네이버 API 키 발급

1. https://developers.naver.com 접속 → 로그인
2. 상단 **Application → 애플리케이션 등록**
3. 애플리케이션 이름: 아무거나 (예: GLN모니터링)
4. 사용 API → **검색** 체크
5. 환경 추가 → **WEB** → URL: `http://localhost`
6. 등록 완료 → **Client ID**, **Client Secret** 복사해두기

---

## 2단계 — Anthropic API 키 발급 + 크레딧 충전

1. https://console.anthropic.com 접속 → 로그인
2. 좌측 **API Keys** → **Create Key** → 키 복사 (sk-ant-... 로 시작)
3. 좌측 **Billing** → 카드 등록 → $5 충전
   - 게시글 1건 분석 = 약 4원, $5면 약 1,500건 분석 가능

---

## 3단계 — Gmail 앱 비밀번호 발급 (이메일 리포트용)

1. https://myaccount.google.com/apppasswords 접속
2. 앱 이름 입력 (예: GLN모니터링) → **만들기**
3. 16자리 비밀번호 복사해두기
   - ⚠️ 일반 Gmail 비밀번호 아님! 앱 비밀번호 따로 만들어야 함

---

## 4단계 — .env 파일에 키 입력

터미널 열고:

```
cd ~/Desktop/GLN카페\ 모니터링/gln-monitor
open .env
```

텍스트 편집기가 열리면 아래 항목 입력 후 **Command + S** 저장:

```
NAVER_CLIENT_ID=발급받은_네이버_ID
NAVER_CLIENT_SECRET=발급받은_네이버_Secret
ANTHROPIC_API_KEY=sk-ant-발급받은_키

SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=내Gmail주소@gmail.com
SMTP_PASS=앱비밀번호16자리
REPORT_FROM=내Gmail주소@gmail.com
REPORT_TO=리포트받을이메일@gmail.com
```

---

## 5단계 — 실행

```
cd ~/Desktop/GLN카페\ 모니터링/gln-monitor
python3 app.py
```

브라우저에서 http://localhost:5001 접속

## 매일 사용할 때

터미널 열고 아래 두 줄만 입력:

```
cd ~/Desktop/GLN카페\ 모니터링/gln-monitor
python3 app.py
```

- 실행하면 즉시 수집 시작
- 1시간마다 자동 수집
- 매일 오전 8시 이메일 리포트 자동 발송
- 중요도 7점 이상 or 부정 감성 감지 시 즉시 알림 이메일

## 자주 나오는 오류

| 오류 | 원인 | 해결 |
|---|---|---|
| 401 Unauthorized | 네이버 API 키 오류 | .env에 키 다시 확인 |
| credit balance too low | Anthropic 크레딧 없음 | console.anthropic.com에서 충전 |
| Port 5000 in use | AirPlay가 포트 점유 | localhost:5001 로 접속 |
| No such file or directory | 폴더 경로 틀림 | cd 명령어 경로 확인 |
